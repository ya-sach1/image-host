Dapper Image Host

A privacy-conscious, configurable image host

Your files are yours

Your files can be securely encrypted, preventing any unauthorized user (even the server owner) from reading your files.

The freedom to experiment

You can control nearly anything about the image host - domains, subdomains, upload names, or even the embed of your image.

A community of care

Dapper Image Host is open source on GitHub, so you can request any feature - at any time.

Free to use

Dapper Image Host is absolutely free to use. No payments, no subscriptions, no abstruse invitation systems.
